## 📌 Project Overview

Basic Chatbot is a Python-based rule-driven conversational application designed to simulate a simple human-like interaction.

The chatbot responds to predefined user inputs using conditional logic and programmed responses. This project demonstrates the fundamental concepts behind chatbot development, including user input processing, response generation, and continuous conversation flow.

---

## ✨ Features

- 💬 Interactive conversation
- 🧠 Predefined intelligent responses
- 🔄 Continuous user interaction
- ⌨️ User input processing
- 🚪 Exit command support
- 💻 Simple console-based interface

## 🚀 How To Run

### Step 1: Install Python

Make sure Python 3 is installed on your system.

### Step 2: Run the Application

Open the terminal in the project folder and execute:

```bash
python chatbot.py
```

### Step 3: Start Chatting

- Enter your messages in the console.
- The chatbot will respond based on predefined rules.
- Type the exit command to end the conversation.
